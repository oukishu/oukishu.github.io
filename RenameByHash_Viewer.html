RenameByHash.ps1

PowerShell Script · Rename files based on SHA256 hashes and automatically deduplicate

📖 Instructions

Features

This script iterates through files with a specific extension in a designated directory, calculates each file's SHA256 hash value, and renames it using the last 12 characters of that hash. If identical duplicate files are found, the script automatically deletes the copies, keeping only the first one.

Prerequisites

  • Windows OS, PowerShell 5.1+ or PowerShell 7+
  • Read/Write permissions for the target directory
  • Execution policy adjustments may be required for the first run (see below)

How to Run

Method 1: Interactive Execution (The script will prompt you step-by-step)

.\RenameByHash.ps1

Method 2: Run directly with parameters

.\RenameByHash.ps1 -TargetDir "C:\Photos" -Extension "jpg"

Method 3: Current directory, default extension (png)

.\RenameByHash.ps1 -TargetDir "" -Extension ""

Parameter Breakdown

  • -TargetDir — The target folder path. Leave blank for interactive input; if left blank during prompts, it defaults to the current directory.
  • -Extension — File extension, e.g., png, jpg (no dot needed). Defaults to png if left blank.

Output Indicators

  • Green — File successfully renamed
  • Red — Duplicate file deleted
  • Yellow — No matching files found in the directory

Execution Policy (First-time use)

If you receive an error stating that the script cannot be run, open PowerShell as an Administrator and execute:

Set-ExecutionPolicy -Scope CurrentUser RemoteSigned

⚠️ Notes

  1. The script will permanently delete detected duplicate files. Please back up important data before operating.
  2. Renaming uses the last 12 digits of the hash. While collisions are extremely rare, the script includes automatic suffix handing (-1, -2...).
  3. It only processes files in the current directory and does not recursively search subfolders.
📄 RenameByHash.ps1 
# Parameters definition
param(
    [string]$TargetDir,
    [string]$Extension
)

# Interactive path retrieval
if ([string]::IsNullOrWhiteSpace($TargetDir)) {
    $TargetDir = Read-Host "Enter target folder path (Press Enter for current directory)"
}
if ([string]::IsNullOrWhiteSpace($TargetDir)) { $TargetDir = Get-Location }

# Interactive extension retrieval
if ([string]::IsNullOrWhiteSpace($Extension)) {
    $Extension = Read-Host "Enter file extension (e.g., png, jpg)"
}
if ([string]::IsNullOrWhiteSpace($Extension)) { $Extension = "png" }
if (!$Extension.StartsWith(".")) { $Extension = "." + $Extension }

Write-Host "`n--- Task Started ---" -ForegroundColor Cyan
Write-Host "Directory: $TargetDir"
Write-Host "Extension: $Extension"

if (!(Test-Path $TargetDir)) {
    Write-Host "Error: Folder does not exist!" -ForegroundColor Red
    pause; exit
}

# Get list of files
$files = Get-ChildItem -Path $TargetDir -Filter "*$Extension" -File

if ($files.Count -eq 0) {
    Write-Host "No matching files found." -ForegroundColor Yellow
} else {
    # Hash Map to store discovered hashes for duplicate detection
    $hashMap = @{}

    foreach ($f in $files) {
        try {
            # Calculate full SHA256 Hash
            $hash = (Get-FileHash $f.FullName -Algorithm SHA256).Hash.ToUpper()
            
            # --- Deduplication Logic ---
            if ($hashMap.ContainsKey($hash)) {
                # If hash exists, it is a duplicate
                $originalFile = $hashMap[$hash]
                Remove-Item -Path $f.FullName -Force
                Write-Host "Deleted Duplicate: $($f.Name) (Identical to $originalFile)" -ForegroundColor Gray
                continue
            }

            # Record new hash in the map
            $hashMap[$hash] = $f.Name

            # --- Renaming Logic ---
            # Use the last 12 characters of the hash as the new filename
            $newNameBase = $hash.Substring($hash.Length - 12)
            $newName = $newNameBase + $Extension
            
            # Skip if the file is already named correctly
            if ($f.Name -eq $newName) { continue }

            # Conflict handling (low probability collision check)
            if (Test-Path (Join-Path $TargetDir $newName)) {
                $count = 1
                while (Test-Path (Join-Path $TargetDir "$newNameBase-$count$Extension")) {
                    $count++
                }
                $newName = "$newNameBase-$count$Extension"
            }

            Rename-Item -Path $f.FullName -NewName $newName -ErrorAction Stop
            Write-Host "Renamed: $($f.Name) -> $newName" -ForegroundColor Green
            
            # Update the map with the new filename
            $hashMap[$hash] = $newName
        }
        catch {
            Write-Host "Failed to process: $($f.Name). Reason: $($_.Exception.Message)" -ForegroundColor Red
        }
    }
}

Write-Host "`n--- Task Completed ---" -ForegroundColor Cyan
pause
